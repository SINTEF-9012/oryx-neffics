loadModel

void loadModel ( uri , opt )
Loads the specified Oryx model in the viewer.
Parameters:
uri <String> The URI of the model (without /self). For example: 'http://oryx-editor.org/backend/poem/model/1234'
opt <Object> Options:
onSuccess
Callback to execute when the model is finished loading The callback receives the ModelViewer instance back.
onFailure
Callback to execute when the model load operation fails The callback receives the ModelViewer instance back.
onTimeout
Callback to execute when a timeout occurs. If not set the onFailure callback will be executed on timeout. The callback receives the ModelViewer instance back.
scope
The execution scope for the callbacks.
timeout
Number of milliseconds to wait before aborting the loading of the model and executing the onFailure callback. The default value is 15000 (15 seconds).
Returns: void

namespace

static Object namespace ( arguments )
Returns the namespace specified and creates it if it doesn't exist 
MOVI.namespace("property.package");
MOVI.namespace("MOVI.property.package");
Either of the above would create MOVI.property, then MOVI.property.package Be careful when naming packages. Reserved words may work in some browsers and not others. For instance, the following will fail in Safari: 
MOVI.namespace("really.long.nested.namespace");
This fails because "long" is a future reserved word in ECMAScript This method is taken from YUI's YAHOO object.
Parameters:
arguments <String*> 1-n namespaces to create
Returns: Object
A reference to the last namespace object created

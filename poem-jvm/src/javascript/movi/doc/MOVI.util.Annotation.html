Class MOVI.util.Annotation - extends YAHOO.util.Element

Create an Annotation for a Marker to enrich the model with additional information and functionality. The Annotation renders a speech bubble containing arbitrary XHTML content and attachs it to the Marker.

Parameters:
marker <Marker> The Marker to attach the Annotation to
content <String> The Annotation's inner HTML content

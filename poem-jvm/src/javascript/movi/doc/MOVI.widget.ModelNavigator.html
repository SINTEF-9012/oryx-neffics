Class MOVI.widget.ModelNavigator - extends YAHOO.util.Element

The ModelNavigator widget is a UI control that enables users to navigate in a model displayed in a ModelViewer component. A rectangle indicates the model clipping.

Constructor

MOVI.widget.ModelNavigator ( el )
Parameters:
el <HTMLElement | String > The id of the container DIV element that will wrap the ModelNavigator, or a reference to a DIV element. The DIV element must exist in the document.

Properties

modelviewer - ModelViewer

The ModelViewer that is navigated

Methods

update

Callback to update the clipping rectangle's position and size. Also updates the image if another model has been loaded in the model viewer.

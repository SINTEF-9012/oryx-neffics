Class MOVI.util.ShapeSelect

Enbable shape selection for the specified model viewer

Constructor

MOVI.util.ShapeSelect ( modelviewer , shapes , multiselect )
Parameters:
modelviewer <ModelViewer> The ModelViewer for that shape selection is enabled
shapes <Shape*> (optional) The subset of shapes that are selectable. If not specified all shapes are selectable.
multiselect <Boolean> (optional) If set to true, multiple shapes can be selected (default is false).

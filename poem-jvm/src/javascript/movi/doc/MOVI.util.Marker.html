Class MOVI.util.Marker - extends YAHOO.util.Element

Attach Marker objects to the model to highlight a shape or a set of shapes by overlaying rectangles

Constructor

MOVI.util.Marker ( shapes , style )
Parameters:
shapes <Shape|Shapes[]> The shapes to mark
style <Object> (optional) A key map of CSS style properties to be attached to the shape marking rectangles.

Properties

canvas - Canvas

The parent canvas element of this marker

markerRect - Element

The marker's outer rectangle element

MOVI.util.Marker.PADDING - static integer

The padding beween the marker's border and the shape's border.
Default Value: 2

shapeRects - { Integer : Element }

A key map containing all shape rectangle elements of the marker with the associated shape resource IDs as keys
